# BMI-and-Glycaemic-Traits-Mendelian-Randomization
This repository includes scripts to reproduce the main results presented in the following manuscript:
Causal effects of body mass index on glycaemic traits in African and European populations: a Mendelian randomization study

In the following sections, we describe step by step how to reproduce the results. 
The scripts are placed into three directories. 
Please enter into each of these directories and run the scripts according to the description described below. 
The prsented scripts are meant for reproduction of the results in the paper.
